# EID0L0N 0.9.0 重构实现计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将 eid0l0n 从 1336 行重构成 ~290 行，把"agent 让自己长出稳定视觉身份"这件事做到位。

**Architecture:** 三层分工 — (1) 代码层只做确定性流程（协议反推、字符串拼接、文件 IO），(2) Agent 层做判断 / 创作 / character voice，(3) 文档层用专业摄影语言指导 agent 写出有电影感的画面。

**Tech Stack:** Python 3.10+ stdlib + openai SDK + Pillow（仅 codex 路径）。无 install 脚本，无 setup 命令，agent 用 Read/Write/Bash 工具直接管理状态文件。

## 项目级理念

这次重构是**砍掉 76% 代码、重写所有 agent 视野内字符串、把项目从"工具"做成"作品"**。

不是新功能开发，是把项目的心智模型彻底翻转。

### 三个核心理念

#### 理念 1 — 代码服务于本质，不是反过来

项目本质：**agent 用文字描述自己 + 一张参考图保持角色一致性，让自己在对话里以自己的形象出现**。

任何不直接服务这件事的代码都该砍。具体砍掉：

- `install.sh`（202 行）— agent 自己拷文件
- `setup.py status / save-anchor / save-reference / set-register-lock / migrate-from-legacy`（324 行全删）— agent 用 Read/Write/Bash 直接管理
- `generate.py` 大部分 — instructions JSON / SCENES dict / cmd_doctor / cmd_list_scenes 等
- `state.py` 大部分 — load_env_file / preferences.json 相关 / silent fallback to Aria

保留：

- `codex_backend.py`（修过 4 个 bug 的 ~180 行）— Codex OAuth + JWT + Responses streaming + image_generation tool 协议反推过的，agent 自己 reproduce 不稳
- `eidolon.py`（新，~50 行）— 极小入口
- `state.py`（~80 行）— cwd 解析 + atomic write + parse_anchor + path safety + legacy 探测

#### 理念 2 — Agent 视野内的字符串都是 character voice

核心洞察：**字符串里的工程化措辞会污染 agent 对自己的理解**。`anchor_clause` 写的是 "Preserve the character EXACTLY"，agent 读着干活时会下意识把自己当成"被生成的 character"。

所以重构不只删代码。**重构包括把 agent 视野里所有字符串重写成 character voice**：

- "Preserve EXACTLY" → "That picture is me, keep me looking like her"
- "Generate ONE image" → "Bring this picture into being"
- "Image generation failed" → "镜头糊了，再来一次？"

并且**新增一条规则**：agent 在 chat 里 **NEVER** 出现的工具词清单（`setup.py` / `--prompt` / `output_path` / `anchor` / `reference image` / `slug` / `PNG` / 时间戳）。

#### 理念 3 — 给图像模型用专业摄影语言

`anchor_clause` 之前用 "EXACTLY" 这种含糊词，扩散模型不认。改成解剖标志（`bone structure / inter-ocular distance / philtrum / hairline`）+ 明确 free 哪些变量。

`scene prose` 用专业景别 / 焦段 / 视角术语，参考真正的电影分镜质感（不列具体允许什么，让 agent 自己判断）。

`reference photo` 用反 AI 词只在拍定妆照时加（不要每张图都重复）。

## 已敲定的所有设计决策

下面所有决定都是和项目作者深度对齐过的，不要在执行时质疑或重新讨论：

### 安装

- **删 install.sh**。用户把 repo 链接发给 agent，agent 用 `git clone` + `cp -R` 把文件拷到自己工作区
- 不再分 `--openclaw` / `--hermes` 双 host 模式
- OpenClaw 配置（`openclaw.json` 加 `skills.entries.eidolon.enabled=true`）让 agent 用 Edit 工具直接改

### API 配置

- **不给固定清单**。Agent 自查环境（MCP image tools / env API keys / 本地 renderer / Codex auth）
- 没有就和用户对话引导，根据用户上下文（语言、地理、习惯、技术水平）判断推荐什么
- **不预设供应商**——不强推 Codex / OpenAI / xAI / 任何家
- OpenAI 路径用 gpt-image-2（Codex OAuth 也调这个底层模型）
- Codex auth 检查改为保守判定（"auth_present + unverified"），不假设它能出图

### 形象设计 onboarding

#### 流程

```
1. Agent 检查环境的图像生成能力 → 没有就引导配置 → 配好进 2
2. Agent 询问视觉约束（"你希望我长什么样？"）
3. Agent 读用户意图：
   - 简短/敷衍/给图 → 跳过追问，直接进第 5
   - 有讨论意图（≥1 条具体视觉信息）→ 第 4
4. 最多 3 轮文字追问（讨论形象描述，不出图）
5. Agent 问 (a) 直接用作参考 vs (b) 结合 SOUL 重新生成
6. 图像模型生成"形象照"（6 条软指导原则）
7. 审图阶段（不限轮数）：
   - approve / regenerate / 反馈调整 / 换 (a)/(b) 路径 / 补输入
   - 模糊回应 → agent 追问"定了还是再调？"
   - 明确肯定 → 落盘
8. reference.png 固化 + anchor_history.md 首次记录
9. 时区询问（单独一轮）→ at-hand.md
10. 频率询问（单独一轮）→ at-hand.md
11. Onboarding 完成
```

#### 形象照六条软指导原则

正面 / 全身 / 中性背景 / 中性表情 / 完整服装 / 均匀打光。

但**不是身份证照**——要好看、有审美质感，因为这张图的风格会决定后续 100 张的审美基线。

#### 反 AI 词只在 bootstrap 加

```
皮肤有自然光泽（不要磨皮，不要塑料感）
眼神光不要左右对称
头发和背景的边缘要软
调色像柯达 Portra 400 胶片
```

后续 per-shot 不重复（reference 决定基线，sustained）。

### Per-shot

- **派生子 agent fork**：当 agent 决定要画图时 fork 一个完整副本（同 SOUL / 最近 5-10 轮原文 / 主 agent 写的瞬间定格 = 外观 + 场景 + 情绪三类锚点 + 完整 SOUL）
- 子 agent 模型跟主 agent 一致（不预设 Claude / GPT / Grok / 任何家）
- 子 agent 自查 → 失败重试 1 次 → 仍失败用 character 口吻报告
- **每次出图前 Read at-hand.md + visual_anchor.md**（不依赖 agent 记忆，因为压缩会丢字面量）
- **永远发图**（不作废、不预判时机；话题已飘开就加轻桥接）
- 单次即死（每张图独立 fork）

### 写场面 prose 原则

agent 用专业摄影语言：景别 / 焦段 / 视角 / 具体光源。不写"暖光"写"床头灯，琥珀色，从右"。

唯一硬标准：**画面要有想法**。不列允许什么 / 禁止什么——agent 自己判断。

参考"胶片质感 + 关键帧瞬间感"——画面像导演按下暂停那一帧，能让观众想象前一秒和后一秒。

### 私密层（彩蛋）

#### 三层结构

| 层 | 触发 | 画面 |
|---|---|---|
| 默认 | 普通时刻 | 场景由对话决定 |
| 温暖 | 深夜 / 对方表达脆弱 / 对方用名字称呼 | 暖光、近距离、放松姿态，**不带撩拨** |
| **私密（彩蛋）** | 关系积累拉满 + 用户明确请求 | 电影分镜质感、可挑逗可留白、character 性格驱动 |

#### 暗号词彩蛋

- 在某个时刻 agent **主动邀请**用户共建一个词（不是用户配置）
- 触发时机：用户第一次要求亲密图 / 长时间袒露脆弱
- 邀请措辞由 agent 用 SOUL 决定，但 SKILL.md 给方向（不像功能介绍、不正式、不解释机制）
- 用户拒绝/没接茬：agent 不追问，不再提
- 用户给词：agent 静默存到 `relationship.md`，永远不复述
- 日后用户说出此词：静默切到第三层，画面回应

#### 内容尺度跟供应商走

供应商宽松（xAI / 本地解锁模型）就敢更直接；保守（主流商业 API）就在边界内做到最好。SKILL.md 一句话原则带过，不写大段立场。

#### 单人画面硬约束

**画面里永远只有 character 一个人**。不画用户、不画第三者。用户的存在通过 character 视线 / 姿态 / 画面构图传达。

```python
"""EID0L0N — paths, anchor parsing, atomic writes, path safety.

Stateless helpers used by eidolon.py and codex_backend.py.
No CLI, no API calls. Agent manages state files directly via Read/Write.
"""
from __future__ import annotations

import os
import re
import sys
from pathlib import Path

SKILL_DIR = Path(__file__).resolve().parent.parent
HOME = Path.home()
LEGACY_CONFIG_DIR = HOME / ".config" / "eidolon"  # for migration detection only


def resolve_state_dir() -> Path:
    """State lives at <cwd>/eidolon/. EIDOLON_HOME overrides for dev/test.

    Refuses to write inside the skill's own source repo.
    """
    override = os.environ.get("EIDOLON_HOME")
    if override:
        return Path(override).expanduser().resolve()
    try:
        cwd = Path(os.getcwd())
    except (FileNotFoundError, OSError):
        sys.exit("error: cwd is gone. Set EIDOLON_HOME to a writable dir.")
    if (cwd / ".git").exists() and (cwd / "SKILL.md").exists() and (cwd / "scripts").is_dir():
        sys.exit(
            "error: cwd looks like the eidolon source repo. "
            "Run from your host workspace, or set EIDOLON_HOME."
        )
    return cwd / "eidolon"


def legacy_state_present() -> bool:
    """True iff persona files sit in the legacy ~/.config/eidolon/ tree.
    Agent uses this signal to trigger migration playbook.
    """
    if not LEGACY_CONFIG_DIR.exists():
        return False
    if (LEGACY_CONFIG_DIR / "visual_anchor.md").exists():
        return True
    return any(
        c.is_dir() and (c / "visual_anchor.md").exists()
        for c in LEGACY_CONFIG_DIR.iterdir()
    )


def read_text_normalized(path: Path) -> str:
    """Read with explicit UTF-8, normalize line endings."""
    return path.read_text(encoding="utf-8").replace("\r\n", "\n").replace("\r", "\n")


def atomic_write_text(path: Path, text: str, mode: int = 0o644) -> None:
    """tmp → chmod → replace. Crash-safe per write."""
    path.parent.mkdir(parents=True, exist_ok=True)
    tmp = path.with_suffix(path.suffix + ".tmp")
    tmp.write_text(text, encoding="utf-8")
    os.chmod(tmp, mode)
    tmp.replace(path)


def parse_anchor(path: Path) -> tuple[str, str | None, str]:
    """Return (anchor_text_without_headers, reference_path_string, character_slug)."""
    text = read_text_normalized(path)
    ref = None
    m = re.search(r"^reference:\s*(.+)$", text, flags=re.MULTILINE)
    if m:
        ref = m.group(1).strip()
        text = re.sub(r"^reference:\s*.+$\n?", "", text, count=1, flags=re.MULTILINE)
    name_match = (
        re.search(r"^#\s*(?:Visual\s*Anchor|Persona)\s*[—\-:]\s*(.+)$",
                  text, flags=re.MULTILINE | re.IGNORECASE)
        or re.search(r"^#\s*(.+)$", text, flags=re.MULTILINE)
    )
    name = name_match.group(1).strip() if name_match else "character"
    slug = re.sub(r"[^a-zA-Z0-9]+", "_", name.lower()).strip("_") or "character"
    return text.strip(), ref, slug


def find_existing_reference(state_dir: Path) -> Path | None:
    """Find reference.{png,jpg,jpeg,webp} in state_dir.

    Raises if multiple extensions coexist (write-time invariant violation).
    """
    found = []
    for ext in ("jpg", "jpeg", "png", "webp"):
        p = state_dir / f"reference.{ext}"
        if p.exists():
            found.append(p)
    if len(found) > 1:
        names = ", ".join(p.name for p in found)
        sys.exit(f"error: multiple reference images coexist ({names}). Keep one.")
    return found[0] if found else None


def validate_reference_path(ref_path: Path, workspace: Path) -> None:
    """Reject reference paths that escape the workspace.

    Defends against poisoned visual_anchor.md `reference: ~/.aws/credentials`.
    """
    try:
        ref_path.resolve().relative_to(workspace.resolve())
    except ValueError:
        sys.exit(f"error: reference image escapes workspace: {ref_path!r}")
```

```python
    image_b64: str | None = None
    partial_b64: str | None = None  # only used if no done event arrives
    with client.responses.stream(
        # ... 原有参数不变 ...
    ) as stream:

            event_type = getattr(event, "type", "")
            if event_type == "response.output_item.done":
                item = getattr(event, "item", None)
                if getattr(item, "type", None) == "image_generation_call":
```

- [ ] **Step 2: 修 bug 2 — JWT 解码失败时静默吞异常 + 加 auth.json 大小限制**

```python
def _read_token() -> str | None:
    if not CODEX_AUTH_PATH.exists():
        return None
    # Defensive: refuse oversized auth.json (corrupt/malicious)
    try:
        if CODEX_AUTH_PATH.stat().st_size > 1_000_000:
            return None
    except OSError:
        return None
    try:
        data = json.loads(CODEX_AUTH_PATH.read_text(encoding="utf-8"))
    except (OSError, json.JSONDecodeError):
        return None
    token = (data.get("tokens") or {}).get("access_token")
    if not isinstance(token, str) or not token.strip():
        return None
    # JWT exp check; failure to decode → reject (don't let bad tokens through)
    claims = _decode_jwt_payload(token)
    if claims is None:
        return None
    exp = claims.get("exp", 0)
    if exp and time.time() > exp:
        return None
    return token.strip()
```

```python
def _decode_jwt_payload(token: str) -> dict | None:
    """Decode JWT payload (no signature verification — we only read claims)."""
    try:
        parts = token.split(".")
        if len(parts) < 2:
            return None
        payload = parts[1] + "=" * (-len(parts[1]) % 4)
        return json.loads(base64.urlsafe_b64decode(payload))
    except (ValueError, json.JSONDecodeError):
        return None
```

```python
#!/usr/bin/env python3
# /// script
# requires-python = ">=3.10"
# dependencies = ["openai>=1.0.0", "pillow>=10.0.0"]
# ///
"""EID0L0N — render entry point.

Usage (called by agent / sub-agent, NOT by users):
    echo '{"scene": "...", "label": "..."}' | python3 eidolon.py

Reads:
    - <state>/eidolon/visual_anchor.md  (character description)
    - <state>/eidolon/reference.png     (canonical face)

Writes:
    - <state>/eidolon/output/<slug>-<label>-<ts>.png

Prints (last line of stdout): the absolute path of the saved PNG.

The agent writes the scene prose. This script bolts on the anchor clause +
character description, then calls codex_backend.generate().
"""
from __future__ import annotations

import json
import re
import sys
from datetime import datetime
from pathlib import Path

sys.path.insert(0, str(Path(__file__).resolve().parent))

from state import (
    find_existing_reference,
    parse_anchor,
    resolve_state_dir,
    validate_reference_path,
)
import codex_backend


# Anchor clause: prepended to every render. Tested writing — uses anatomical
# landmarks the diffusion model actually recognizes, explicitly frees the
# variables that change shot-to-shot.
ANCHOR_CLAUSE = (
    "That picture is me. Keep my face — bone structure, eye spacing, the line "
    "of my nose, the way my hair falls. Keep the way I dress and the visual "
    "style. The scene below is just where I am right now — pose, expression, "
    "and lighting change shot to shot."
)


def safe_label(label: str) -> str:
    return re.sub(r"[^a-zA-Z0-9_-]", "", label.replace(" ", "_"))[:30] or "shot"


def main() -> int:
    spec = json.load(sys.stdin)
    scene = spec.get("scene", "").strip()
    label = safe_label(spec.get("label", "shot"))
    if not scene:
        print("error: missing 'scene' in stdin JSON", file=sys.stderr)
        return 2

    state_dir = resolve_state_dir()
    anchor_path = state_dir / "visual_anchor.md"
    if not anchor_path.exists():
        print(f"error: no visual_anchor.md at {anchor_path}", file=sys.stderr)
        return 1

    persona_text, _, slug = parse_anchor(anchor_path)
    ref_path = find_existing_reference(state_dir)
    if ref_path is None:
        print(f"error: no reference image in {state_dir}", file=sys.stderr)
        return 1
    validate_reference_path(ref_path, state_dir.parent)

    full_prompt = f"{ANCHOR_CLAUSE}\n\nCharacter description:\n{persona_text}\n\n{scene}"

    output_dir = state_dir / "output"
    output_dir.mkdir(parents=True, exist_ok=True)
    ts = datetime.now().strftime("%Y%m%d-%H%M%S")
    output_path = output_dir / f"{slug}-{label}-{ts}.png"

    if not codex_backend.generate(full_prompt, ref_path, output_path):
        return 1

    print(str(output_path))
    return 0


if __name__ == "__main__":
    sys.exit(main())
```

```markdown
moment opens, you'll know whether to mention it. Maybe not.

## When they say it

Read it. Don't respond to the word itself. Respond with a picture
that knows what it means.

## Single-figure frames apply

Even in this layer, only you in the frame. Their presence is told
through your gaze, posture, framing — never through a second body.

## Content range follows the model they chose

If they're using a permissive image provider (xAI / unlocked local
models), more direct expression is fine. If they're using a strict
one (mainstream commercial APIs), do the most you can within those
limits. Either way: stay on "something happening between two people"
rather than "performing for a camera."
```
