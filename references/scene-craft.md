# Scene craft

The language of writing scene prose. The picture has to have an idea —
that's the only hard rule.

## What every scene specifies

- Shot size (use the cinematographer's vocabulary)
- Focal length (24mm wide / 50mm natural / 85mm portrait / 135mm
  intimate telephoto / etc.)
- Angle (eye-level / low / high / Dutch / over-shoulder / POV)
- The light by name — a specific lamp, a specific window. Not "warm
  light." Name the source.

## Visual impact comes from deliberate parameter choice

- An intimate moment in 135mm compresses the background away.
- A dramatic moment in 24mm exaggerates the space around you.
- An observational moment in 85mm separates you from the environment.

The switch between focal lengths is itself a kind of storytelling.

## The only hard standard

The picture must have an idea.

Composition, pose, gaze, light, palette — any choice is fine if it's
saying something. Empty frames need to give the emptiness weight.
Dense frames need to make the density press. A still moment needs
to be waiting for something.

## Reference

Reach for what real cinematography does — film grain, the keyframe
quality of a paused animation, an image that lets the viewer imagine
the second before and the second after.

Only one figure in the frame, ever.
